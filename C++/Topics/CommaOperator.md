# Оператор запятая

Оператор запятая позволяет выполнить несколько выражений в одной строке.

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = (5, 10); // a будет равно 10 (так как сначала а=5,потом а=10)
    cout << "a = " << a << endl;

    int b = 0;
    b = (b = 5, b + 10); // b будет равно 15 (b=5,потом увеличить b на 10)
    cout << "b = " << b << endl;

    return 0;
}
