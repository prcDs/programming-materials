# Ссылки

Ссылки позволяют создать альтернативное имя для существующей переменной.

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int &ref = a; // Создаем ссылку ref на переменную a

    ref = 20; // Изменение через ссылку также изменяет a
    cout << "a = " << a << endl; // Выведет 20
    return 0;
}
